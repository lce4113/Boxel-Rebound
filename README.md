# boxel-rebound

Javascript Game influenced by Boxel Rebound  

Play on local web server: [http://localhost:9001/]  

## Start Local Web Server
Open terminal and type the following from the project directory
```term
docker-compose up -d
```

## Stop Local Web Server
Open terminal and type the following from the project directory
```term
docker-compose down
```
